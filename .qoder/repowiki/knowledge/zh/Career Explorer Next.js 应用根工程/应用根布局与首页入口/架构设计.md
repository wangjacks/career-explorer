本模块是 Next.js App Router 的顶层入口：
- `layout.tsx` 作为 RootLayout，通过 `next/font/google` 注入 Geist Sans/Mono 并暴露 CSS 变量，同时挂载 `ErrorBoundary` 与 `InstallGuard` 两个包裹组件，统一处理 PWA 安装引导与渲染期错误；导出 `viewport` 与 `metadata` 配置移动端视口与 SEO 信息。
- `page.tsx` 为站点默认路由（`/`），声明 `dynamic = "force-dynamic"` 强制服务端动态渲染以绕过 SSG 缓存，内部仅组合 `NavigationBar` 与一个指向 `/student` 的 `Link` 作为着陆页。
- `globals.css` 使用 Tailwind v4 的 `@import "tailwindcss"` + `@theme inline` 语法将 CSS 变量映射到 Tailwind 设计令牌，并设置 body 基础样式。
依赖方向单向向下：布局 → 全局样式；页面 → 共享组件（`NavigationBar`）；无同级文件互相引用。
