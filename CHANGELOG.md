# Changelog

## [1.0.0] - 2026-07-09

### Added

- 职业标签选择（兴趣/技能/性格三维度）
- 自定义标签输入
- 实时词云可视化展示
- 评价词云上传页面
- 头像上传与预览
- 学号验证（12 位数字）
- 已有填写记录检测与恢复
- 表单状态 localStorage 持久化
- 管理员面板（登录/数据概览/数据源设置/学生管理/数据导出）
- 数据导出（Excel/CSV，支持筛选/列选择/图片打包）
- 批量学生导入（标题行自动识别）
- 首次安装引导页面
- MySQL 数据库支持
- 数据源配置（SQLite/MySQL 切换）

### Changed

- 从微信小程序重构为 Next.js Web 应用
- 数据库从 SQLite 迁移至 MySQL
- 管理页面改为 Tab 导航

### Fixed

- 提交时间未加时区偏移
- 生产模式下上传图片无法显示
- 恢复记录后跳转页面错误
- 重复上传同名图片时浏览器缓存未更新
